## 5.1 全局参数

### 头寸调整开关

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `position_adjustment_enable` | bool | True | 启用头寸调整 |
| `grinding_enable` | bool | True | 启用磨削 |
| `derisk_enable` | bool | True | 启用风险撤离 |
| `stops_enable` | bool | True | 启用止损 |
| `doom_stops_enable` | bool | True | 末日止损 |

```python
position_adjustment_enable = True
grinding_enable = True
derisk_enable = True
stops_enable = True
doom_stops_enable = True
```

### 期货参数

| 参数 | 类型 | 默认值 | 范围 | 说明 |
|------|------|--------|------|------|
| `futures_mode_leverage` | float | 3.0 | 1-5 | 期货杠杆 |
| `max_slippage` | float | 0.01 | 0.001-0.05 | 最大滑点 |

```python
futures_mode_leverage = 3.0
max_slippage = 0.01
```

### 性能参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `process_only_new_candles` | bool | True | 仅处理新K线 |
| `num_cores_indicators_calc` | int | 0 | 多核指标计算(0=禁用) |
| `startup_candle_count` | int | 800 | 启动蜡烛数 |

```python
process_only_new_candles = True
num_cores_indicators_calc = 0
startup_candle_count = 800
```

### 费用参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `custom_fee_open_rate` | float | 0.0 | 自定义开仓费 |
| `custom_fee_close_rate` | float | 0.0 | 自定义平仓费 |

```python
custom_fee_open_rate = 0.0
custom_fee_close_rate = 0.0
```

## 5.2 模式特定参数

### 仓位乘数

| 参数名称 | 默认值 | 适用模式 |
|----------|--------|----------|
| `regular_mode_stake_multiplier_spot` | 1.0 | Normal |
| `regular_mode_stake_multiplier_futures` | 1.0 | Normal |
| `rebuy_mode_stake_multiplier` | 0.35 | Rebuy |
| `rapid_mode_stake_multiplier_spot` | 0.75 | Rapid |
| `grind_mode_stake_multiplier_spot` | [0.5,0.5,0.5] | Grind |
| `pump_mode_stake_multiplier` | 0.5 | Pump |
| `quick_mode_stake_multiplier` | 0.5 | Quick |

```python
# Normal 模式
regular_mode_stake_multiplier_spot = 1.0
regular_mode_stake_multiplier_futures = 1.0

# Rebuy 模式
rebuy_mode_stake_multiplier = 0.35

# Grind 模式
grind_mode_stake_multiplier_spot = [0.5, 0.5, 0.5]
grind_mode_stake_multiplier_futures = [0.5, 0.5, 0.5]

# Rapid 模式
rapid_mode_stake_multiplier_spot = 0.75

# Pump 模式
pump_mode_stake_multiplier = 0.5

# Quick 模式
quick_mode_stake_multiplier = 0.5
```

## 5.8 配置调优建议

### 保守配置（新手推荐）

```python
# 降低风险
grinding_v1_max_stake = 0.5
regular_mode_derisk_spot = -0.15
rebuy_mode_stake_multiplier = 0.25
grind_mode_max_slots = 2
futures_mode_leverage = 1.0

# 仅启用 Normal 模式
```

### 平衡配置

```python
# 默认参数
grinding_v1_max_stake = 1.0
regular_mode_derisk_spot = -0.24
rebuy_mode_stake_multiplier = 0.35
grind_mode_max_slots = 3
futures_mode_leverage = 2.0

# 启用 Normal + Grind 模式
```

### 激进配置（专家级）

```python
# 增加头寸
grinding_v1_max_stake = 1.5
regular_mode_derisk_futures = -0.35
rebuy_mode_stake_multiplier = 0.50
grind_mode_max_slots = 3
futures_mode_leverage = 3.0

# 启用多模式
```

### 性能优化配置

```python
# 多核处理
num_cores_indicators_calc = 4

# 蜡烛优化
process_only_new_candles = True

# 交易对数量
max_open_trades = 6

# 充足预热
startup_candle_count = 800
```
