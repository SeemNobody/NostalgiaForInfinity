## 3.1 模式分类与标签系统

NFI 使用 `enter_tag` 标签系统来区分和路由不同的交易模式。每笔交易在入场时被分配一个标签，出场时根据此标签调用相应的出场逻辑。

### 标签范围总览

| 模式 | 多头标签 | 空头标签 | 主要用途 |
|------|----------|----------|----------|
| **Normal** | 1-13 | 501-502 | 标准趋势交易 |
| **Pump** | 21-26 | 521-526 | 捕捉快速动量 |
| **Quick** | 41-53 | 541-550 | 短期快速获利 |
| **Rebuy** | 61-63 | 561 | 头寸平均化 |
| **Rapid** | 101-110 | 601-610 | 高频快速入场 |
| **Grind** | 120 | 620 | DCA分层加仓 |
| **Scalp** | 161-163 | 661 | 微利刷单 |
| **Top Coins** | 141-145 | 641-642 | 大市值币种 |
| **BTC** | 121 | N/A | 比特币专用 |

### 标签路由流程

```
入场信号生成 → 分配 enter_tag
       ↓
交易执行 → 记录 enter_tag
       ↓
出场判断 → 根据 enter_tag 路由
       ↓
模式特定出场逻辑执行
```

## 3.2 多头模式详解

### 3.2.1 Normal Long Mode（普通模式）

**标签范围**: 1-13
**适用场景**: 温和波动的标准趋势市场

#### 特点
- 基线趋势跟踪策略
- 多指标确认（RSI, EMA, Aroon, Stochastic RSI）
- 多时间框架分析（5m/15m/1h/4h/1d）
- 30+项保护检查

#### 关键参数
```python
stop_threshold_spot = 0.10              # 现货止损 10%
stop_threshold_futures = 0.10           # 期货止损 10%
regular_mode_stake_multiplier_spot = 1.0  # 仓位乘数
regular_mode_derisk_spot = -0.24        # 风险撤离阈值
```

#### 进场条件示例
- RSI_3 > 30（多时间框架确认）
- EMA-12/26 向上交叉
- 价格位于布林带下轨附近
- BTC 非下跌趋势确认
- MACD 直方图转正

#### 适用建议
- 推荐开仓数: 6-12个
- 风险级别: 低
- 适合新手入门

---

### 3.2.2 Pump Long Mode（泵模式）

**标签范围**: 21-26
**适用场景**: 高波动性、价格突破行情

#### 特点
- 捕捉上升动量
- 放松RSI约束
- 优先考虑成交量和价格加速
- 更激进的进场位置

#### 关键参数
```python
stop_threshold_spot = 0.10
pump_mode_stake_multiplier = 0.5
# 成交量要求: 平均成交量的 4倍以上
# 价格涨幅: 15分钟内 15%以上
```

#### 进场条件
- 成交量 > 4x 平均成交量
- 15分钟价格涨幅 > 15%
- RSI 快速上升
- 突破近期阻力

#### 风险提示
> ⚠️ **警告**: 容易进入"泵-抛"类骗局，需要配合黑名单过滤。

---

### 3.2.3 Quick Long Mode（快速模式）

**标签范围**: 41-53
**适用场景**: 短期动量、快速获利

#### 特点
- 快速进场、快速离场
- 缩短确认时间
- 禁用磨削机制
- 固定获利目标（1-3%）

#### 关键参数
```python
quick_mode_stake_multiplier = 0.5
# 获利目标: 1-3%
# 进场缓冲: 3-5根K线
# 超时设置: 15分钟强制离场
```

#### 适用场景
- 高流动性币种
- 明确的短期趋势
- 快速进出需求

---

### 3.2.4 Rebuy Long Mode（平均成本模式）

**标签范围**: 61-62
**适用场景**: 波动但区间盘整的市场

#### 特点
- 成本平均化策略
- 在亏损 8-10% 时加仓
- 最多 2 次追加进场
- 改善平均进价

#### 关键参数
```python
rebuy_mode_stake_multiplier = 0.35
rebuy_mode_thresholds_spot = [-0.08, -0.10]   # 触发阈值
rebuy_mode_stakes_spot = [1.0, 1.0]           # 追加仓位
rebuy_mode_derisk_spot = -0.24                # 风险撤离
rebuy_mode_min_free_slots = 2                 # 最小空闲槽位
```

#### 追加进场条件
全部条件需同时满足：
- RSI_3 > 10（双时间框架）
- RSI_14 < 40
- ROC_2 > 0
- 价格 < EMA_26 × 0.988

#### 决策流程
```
头寸亏损 (-8%以下)
    ↓
检查开仓订单 → 有 → 跳过
    ↓ 否
检查指标确认
    ↓ 全部通过
计算头寸大小 (1.0x初始)
    ↓
执行 Rebuy 订单
```

---

### 3.2.5 Rapid Long Mode（快速标量模式）

**标签范围**: 101-110
**适用场景**: 超短期、高流动性市场

#### 特点
- 极速交易优先
- MACD 柱形图突增
- 成交量爆涨（3倍平均）
- 非常紧的止损

#### 关键参数
```python
rapid_mode_stake_multiplier_spot = 0.75
stop_threshold_rapid_spot = 0.20    # 20%止损
max_slippage = 0.01                 # 1%最大滑点
```

#### 性能要求
- 低延迟 API 连接
- 高流动性交易对
- WebSocket 数据流
- 靠近交易所数据中心部署

---

### 3.2.6 Grind Long Mode（磨削模式）

**标签范围**: 120
**适用场景**: 缓慢上升趋势、区间震荡

#### 特点
- 分层平均加仓
- 系统性利润获取
- 支撑/阻力水平多点进场
- 部分获利模式

#### 6级磨削配置

| 级别 | 仓位乘数 | 触发阈值 | 利润目标 |
|------|----------|----------|----------|
| 1 | 0.24-0.28 | -12%~-20% | 1.8% |
| 2 | 0.20-0.28 | -12%~-20% | 1.8% |
| 3 | 0.20-0.24 | -12%~-20% | 1.8% |
| 4 | 0.20-0.24 | -12%~-20% | 1.8% |
| 5 | 0.20-0.24 | -12%~-20% | 4.8% |
| 6 | 0.10-0.18 | -3%~-22% | 1.8% |

#### 关键参数
```python
grind_mode_max_slots = 1
grind_1_stakes_spot = [0.24, 0.26, 0.28]
grind_1_sub_thresholds_spot = [-0.12, -0.16, -0.20]
grind_1_profit_threshold_spot = 0.018
grind_1_stop_grinds_spot = -0.50
```

---

### 3.2.7 Scalp Long Mode（刷单模式）

**标签范围**: 161-163
**适用场景**: 超短期、高频交易

#### 特点
- 极小价格波动捕捉（0.5-1.5%）
- 1分钟级RSI背离
- 订单簿失衡分析
- 极速进出

#### 关键参数
```python
min_free_slots_scalp_mode = 1
stop_threshold_scalp_spot = 0.20        # 20%止损
stop_threshold_scalp_futures = 0.20
# 利润目标: 0.5-1.5%
# 最长持仓: 5-10分钟
# 进场确认: 3:1买卖压力比
```

#### 要求
- 超紧密点差 (< 0.05%)
- 高成交量交易对
- 低网络延迟
- 专业级基础设施

## 3.3 空头模式详解

空头模式与多头模式对称，使用相反的技术指标信号。

### 3.3.1 Short Normal Mode（空头普通模式）

**标签范围**: 501-502

#### 关键差异
- 看空 RSI 信号
- 成交量下降确认
- EMA 向下交叉

#### 参数
```python
stop_threshold_spot = 0.10
regular_mode_derisk_futures = -0.60
```

---

### 3.3.2 Short Pump Mode（空头泵模式）

**标签范围**: 521-526

#### 进场条件
- RSI > 70（严重超买）
- 高位成交量
- 价格偏离移动平均线 2倍ATR 以上

#### 配置指导
| 流动性 | ATR阈值 | 说明 |
|--------|---------|------|
| 高 | 1.5-2.0 | 中等阈值 |
| 中 | 2.0-2.5 | 增加阈值 |
| 低 | 3.0+ | 高阈值 |

---

### 3.3.3 Short Quick Mode（空头快速模式）

**标签范围**: 541-550

#### 进场条件
- 空头吞噬烛形
- MACD 翻转
- 订单簿失衡（卖/买 > 1.5）
- 价格在EMA下方

---

### 3.3.4 Short Rebuy Mode（空头重入模式）

**标签范围**: 561

#### 流程
1. 部分离场获利
2. 等待看空动量确认
3. 以更优价格重新进入

#### 配置
```python
rebuy_mode_thresholds_futures = [-0.08, -0.10]
rebuy_mode_stakes_futures = [1.0, 1.0]
rebuy_mode_derisk_futures = -0.60
```

---

### 3.3.5 Short Rapid Mode（空头快速标量模式）

**标签范围**: 601-610

与多头 Rapid 模式对称，专注于空头动量捕捉。

---

### 3.3.6 Short Grind Mode（空头磨削模式）

**标签范围**: 620

#### 进场条件
- RSI_14 > 64（非常看跌）
- RSI_3 < 90
- AROON Down < 25
- 接近50日低点

---

### 3.3.7 Short Scalp Mode（空头刷单模式）

**标签范围**: 661

#### 配置参数
```python
short_scalp_profit_buffer = 0.001   # 0.1%
short_scalp_timeout = 15            # 分钟
short_scalp_min_volume = 1000
short_scalp_min_spread = 0.0005     # 0.05%
short_scalp_profit_target = 0.005   # 0.5%
```

## 3.4 模式选择矩阵

### 根据市场条件选择

| 市场条件 | 推荐多头模式 | 推荐空头模式 | 原因 |
|----------|-------------|-------------|------|
| **强势上升趋势** | Normal + Grind | - | 稳定利润积累 |
| **强势下降趋势** | - | Normal + Grind | 稳定空头利润 |
| **快速突破** | Pump | Short Pump | 抓住爆发行情 |
| **波动盘整** | Rebuy | Short Rebuy | 利用反弹 |
| **高波动** | Rapid + Scalp | Short Rapid/Scalp | 快速捕捉微移动 |
| **缓慢趋势** | Grind | Short Grind | 分层加仓 |
| **保守策略** | Normal | Short Normal | 多层保护 |

### 根据风险偏好选择

| 风险级别 | 推荐配置 | 说明 |
|----------|----------|------|
| **保守** | 仅启用 Normal 模式 | 最低风险 |
| **平衡** | Normal + Grind | 风险收益平衡 |
| **激进** | Quick + Rapid + Scalp | 高频高风险 |
| **专业** | 多模式 + 精细调优 | 需要经验 |

### 模式特征对比

| 特征 | Normal | Pump | Quick | Rebuy | Rapid | Grind | Scalp |
|------|:------:|:----:|:-----:|:-----:|:-----:|:-----:|:-----:|
| **执行速度** | 中 | 快 | 极快 | 中 | 极快 | 中 | 极快 |
| **持仓时间** | 长 | 短 | 极短 | 中 | 极短 | 长 | 极短 |
| **利润目标** | 1-5% | 同标准 | 1-3% | 分阶段 | 5% | 1.8-4.8% | 0.5-1.5% |
| **风险级别** | 低 | 高 | 中 | 中 | 高 | 低 | 高 |
| **磨削支持** | ✅ | ❌ | ❌ | 可选 | ❌ | ✅ | ❌ |
| **适合新手** | ✅ | ❌ | ❌ | ❌ | ❌ | ✅ | ❌ |

## 3.5 启用/禁用特定条件

### 配置方法

所有入场条件都可通过配置启用/禁用：

```python
# 多头条件
long_entry_condition_1_enable = True/False
long_entry_condition_2_enable = True/False
# ... 继续到 long_entry_condition_163_enable

# 空头条件
short_entry_condition_501_enable = True/False
short_entry_condition_502_enable = True/False
# ... 继续到 short_entry_condition_661_enable
```

### JSON 配置示例

```json
{
  "long_entry_signal_params": {
    "long_entry_condition_1_enable": true,
    "long_entry_condition_2_enable": true,
    "long_entry_condition_120_enable": false
  },
  "short_entry_signal_params": {
    "short_entry_condition_501_enable": true,
    "short_entry_condition_661_enable": false
  }
}
```
