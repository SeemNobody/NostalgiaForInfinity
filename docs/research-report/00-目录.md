## 核心要点速览

### 关键信息
- **必需时间框架**: 5分钟 (`timeframe = "5m"`) - **不可修改**
- **启动蜡烛数**: 800
- **交易模式**: 14种（7多头 + 7空头）
- **支持交易所**: 11个主流交易所

### 必需配置
```json
{
  "use_exit_signal": true,
  "exit_profit_only": false,
  "ignore_roi_if_entry_signal": true
}
```

### 推荐设置
- 并发交易数: 6-12个
- 交易对数量: 40-80个
- 干运行时间: 最少2周
- 最小资金: $1000
