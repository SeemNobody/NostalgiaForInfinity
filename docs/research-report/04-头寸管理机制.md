## 4.1 Grinding（磨削/摊低）机制

### 机制概述

Grinding 是 NFI 的核心头寸管理机制，通过在亏损头寸上追加较小的买入来降低平均成本。

### 工作流程

```
亏损头寸检测
    ↓
检查指标确认 (RSI, ROC, EMA)
    ↓
计算追加头寸大小
    ↓
执行摊低订单
    ↓
更新平均成本
    ↓
检查利润目标 → 达到 → 部分/全部出场
    ↓
未达到 → 继续监控下一级别
```

### 全局开关

```python
grinding_enable = True              # 启用磨削
position_adjustment_enable = True   # 启用头寸调整
```

## 4.2 Rebuy（平均成本）机制

### 机制概述

> **重要澄清**: 尽管名称含义为"回购"，Rebuy 实际上是针对**亏损头寸**的受控摊低策略。

### 关键参数

```python
rebuy_mode_min_free_slots = 2             # 最小空闲槽位
rebuy_mode_stake_multiplier = 0.35        # 初始头寸乘数

# 触发阈值
rebuy_mode_thresholds_spot = [-0.08, -0.10]   # -8%, -10%
rebuy_mode_thresholds_futures = [-0.08, -0.10]

# 追加仓位
rebuy_mode_stakes_spot = [1.0, 1.0]           # 两次追加，各1.0x
rebuy_mode_stakes_futures = [1.0, 1.0]

# 风险撤离
rebuy_mode_derisk_spot = -0.60
rebuy_mode_derisk_futures = -0.60
```

### 追加进场条件

全部条件需同时满足：

| 条件 | 阈值 | 说明 |
|------|------|------|
| RSI_3 (5m) | > 10.0 | 非极度超卖 |
| RSI_3_15m | > 10.0 | 15分钟确认 |
| RSI_14 | < 40.0 | 显示弱势但非极端 |
| ROC_2 | > 0.0 | 动量非极度负面 |
| Close | < EMA_26 × 0.988 | 价格低于关键均线 |

### 决策流程图

```
头寸亏损 (-8%以下)
    │
    ▼
┌─────────────────┐
│ 检查开仓订单    │──有──▶ 跳过
└────────┬────────┘
         │ 否
         ▼
┌─────────────────┐
│ 检查指标确认    │──未通过──▶ 跳过
│ (RSI/ROC/EMA)   │
└────────┬────────┘
         │ 全部通过
         ▼
┌─────────────────┐
│ 计算头寸大小    │
│ (1.0x初始)      │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 验证头寸有效性  │──无效──▶ 跳过
│ min_stake×1.55  │
└────────┬────────┘
         │ 有效
         ▼
┌─────────────────┐
│ 执行 Rebuy 订单 │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 更新平均成本    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│ 检查 De-risk    │──触发──▶ 平仓
│ (-60%亏损)      │
└─────────────────┘
```

## 4.3 Derisking（风险撤离）系统

### 机制概述

Derisking 在达到亏损阈值或特定条件时自动减少头寸，保护资本。

### 全局开关

```python
derisk_enable = True
```

### 各模式风险撤离阈值

| 模式 | 现货 | 期货 | 说明 |
|------|------|------|------|
| **Normal** | -24% | -60% | 标准减仓 |
| **Rebuy** | -60% | -60% | 平均化减仓 |
| **Grind v1** | -50% | -50% | 磨削止损 |
| **Rapid** | -24% | -60% | 使用常规参数 |
| **Scalp** | -5% | -5% | 激进减仓 |

### 配置参数

```python
# 常规模式
regular_mode_derisk_spot = -0.24
regular_mode_derisk_futures = -0.60

# Rebuy模式
rebuy_mode_derisk_spot = -0.60
rebuy_mode_derisk_futures = -0.60

# Grind模式
grind_mode_derisk_spot = -0.50
grind_mode_derisk_futures = -0.50

# Scalp模式
scalp_mode_derisk_spot = -0.05
scalp_mode_derisk_futures = -0.05
```

### Derisking 执行逻辑

```python
# 伪代码示例
if profit_stake < (slice_amount * derisk_threshold / leverage):
    sell_amount = trade.amount * exit_rate / leverage - min_stake * 1.55
    return -sell_amount, "derisk", is_derisk
```

## 4.4 止损体系

### 按模式分类

| 模式 | 现货止损 | 期货止损 | 说明 |
|------|----------|----------|------|
| **Normal** | 10% | 10% | 标准止损 |
| **Pump** | 10% | 10% | 标准止损 |
| **Doom** | 20% | 20% | 末日止损 |
| **Rebuy** | 100%* | 100%* | *由Derisk控制 |
| **Rapid** | 20% | 20% | 宽松止损 |
| **Scalp** | 20% | 20% | 宽松止损 |

### 关键参数

```python
stop_threshold_spot = 0.10              # 现货基础止损
stop_threshold_futures = 0.10           # 期货基础止损
stop_threshold_rapid_spot = 0.20        # Rapid模式
stop_threshold_scalp_spot = 0.20        # Scalp模式
doom_stops_enable = True                # 末日止损开关
```

### 止损执行优先级

```
1. 模式特定止损 (Rapid/Scalp: 20%)
    ↓
2. 基础止损 (Normal: 10%)
    ↓
3. Derisking 触发
    ↓
4. 末日止损 (Doom: 20%)
```

## 4.5 头寸管理最佳实践

### 保守配置

```python
grinding_v1_max_stake = 0.5
regular_mode_derisk_spot = -0.15
rebuy_mode_stake_multiplier = 0.25
grind_mode_max_slots = 2
```

### 激进配置（期货 3x杠杆）

```python
grinding_v1_max_stake = 1.5
regular_mode_derisk_futures = -0.35
rebuy_mode_stake_multiplier = 0.50
grind_mode_max_slots = 3
```

### 风险管理建议

| 建议 | 说明 |
|------|------|
| 单笔最大风险 | 不超过账户的 2% |
| 总敞口 | 不超过账户的 30% |
| 并发磨削 | 最多 3 个头寸 |
| Rebuy次数 | 最多 2 次 |
| 定期审查 | 每周检查磨削头寸 |
