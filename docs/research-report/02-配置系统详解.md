## 2.1 配置层级体系

NFI 采用多层配置覆盖系统，后加载的配置优先级更高：

```
优先级递增 ↓

1. recommended_config.json (基础配置)
   ↓
2. trading_mode-spot.json 或 trading_mode-futures.json (交易模式)
   ↓
3. pairlist-volume-binance-usdt.json (交易对列表)
   ↓
4. blacklist-binance.json (黑名单)
   ↓
5. exampleconfig.json (用户自定义)
   ↓
6. exampleconfig_secret.json (密钥配置，最高优先级)
```

### 配置文件加载示例

```json
{
  "strategy": "NostalgiaForInfinityX6",
  "add_config_files": [
    "../configs/trading_mode-spot.json",
    "../configs/pairlist-volume-binance-usdt.json",
    "../configs/blacklist-binance.json",
    "../configs/exampleconfig.json",
    "../configs/exampleconfig_secret.json"
  ]
}
```

### 配置覆盖规则

1. **同名参数**: 后加载的配置覆盖先前的
2. **数组参数**: 完全替换，不合并
3. **嵌套对象**: 深度合并
4. **策略参数**: 通过 `nfi_parameters` 块覆盖

## 2.2 必需配置参数

### 基础交易参数

| 参数 | 推荐值 | 说明 | 必需 |
|------|--------|------|:----:|
| `max_open_trades` | 6-12 | 最大并发交易数 | ✅ |
| `stake_currency` | "USDT" | 结算货币 | ✅ |
| `stake_amount` | "unlimited" | 使用百分比计算 | ✅ |
| `timeframe` | "5m" | **必须**为5分钟 | ✅ |
| `dry_run` | true | 生产前启用干运行 | ✅ |
| `dry_run_wallet` | 10000 | 干运行初始余额 | |
| `stoploss` | -0.99 | 策略内部动态处理 | ✅ |
| `use_exit_signal` | true | 使用出场信号 | ✅ |
| `exit_profit_only` | false | 允许不限利润出场 | ✅ |
| `ignore_roi_if_entry_signal` | true | 入场信号优先 | ✅ |

### 完整基础配置示例

```json
{
  "strategy": "NostalgiaForInfinityX6",
  "max_open_trades": 6,
  "stake_currency": "USDT",
  "stake_amount": "unlimited",
  "timeframe": "5m",
  "dry_run": true,
  "dry_run_wallet": 10000,
  "stoploss": -0.99,
  "use_exit_signal": true,
  "exit_profit_only": false,
  "ignore_roi_if_entry_signal": true
}
```

## 2.3 交易模式配置

### 期货模式

```json
{
  "trading_mode": "futures",
  "margin_mode": "isolated",
  "futures_mode_leverage": 3.0,
  "dataformat_ohlcv": "feather",
  "dataformat_trades": "feather"
}
```

| 参数 | 值 | 说明 |
|------|-----|------|
| `trading_mode` | "futures" | 期货模式激活 |
| `margin_mode` | "isolated" | 逐仓模式（风险隔离） |
| `futures_mode_leverage` | 1-5 | 推荐 1-3x |
| `dataformat_ohlcv` | "feather" | 高效数据存储 |

### 现货模式

```json
{
  "trading_mode": "spot",
  "dataformat_ohlcv": "feather",
  "dataformat_trades": "feather"
}
```

### 杠杆建议

| 经验级别 | 推荐杠杆 | 说明 |
|----------|----------|------|
| 新手 | 1x | 无杠杆 |
| 中级 | 2x | 温和杠杆 |
| 高级 | 3x | 标准杠杆 |
| 专业 | 5x | 高风险 |

## 2.4 交易对列表管理

### 推荐配置

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| **交易对数量** | 40-80对 | 平衡流动性和多样化 |
| **配对方式** | VolumePairList | 按交易量动态排名 |
| **刷新周期** | 1800秒 | 30分钟 |
| **最少上市天数** | 60天 | 避免新币风险 |
| **最低价格比率** | 0.003 | 过滤极低价资产 |
| **最大点差比例** | 0.005 | 排除高滑点交易对 |

### 交易对列表类型

| 类型 | 命名模式 | 用途 |
|------|----------|------|
| **静态列表** | `pairlist-static-{exchange}-{market}-{quote}.json` | 生产环境 |
| **动态列表** | `pairlist-volume-{exchange}-{quote}.json` | 按交易量排名 |
| **回测列表** | `pairlist-backtest-static-{exchange}-{market}-{quote}.json` | 防止前瞻偏差 |

### 动态交易对列表配置

```json
{
  "pairlists": [
    {
      "method": "VolumePairList",
      "number_assets": 80,
      "sort_key": "quoteVolume",
      "refresh_period": 1800
    },
    {
      "method": "AgeFilter",
      "min_days_listed": 60
    },
    {
      "method": "PriceFilter",
      "low_price_ratio": 0.003
    },
    {
      "method": "SpreadFilter",
      "max_spread_ratio": 0.005
    }
  ]
}
```

### 过滤器说明

| 过滤器 | 作用 | 参数 |
|--------|------|------|
| `VolumePairList` | 按交易量排名 | `number_assets`, `refresh_period` |
| `AgeFilter` | 过滤新币 | `min_days_listed` |
| `PriceFilter` | 过滤低价币 | `low_price_ratio` |
| `SpreadFilter` | 过滤高滑点 | `max_spread_ratio` |

## 2.5 黑名单配置

### 被排除的资产类型

| 类别 | 示例 | 原因 |
|------|------|------|
| **交易所代币** | BNB, KCS, OKB | 可能有特殊交易规则 |
| **杠杆代币** | *BULL, *BEAR, *UP, *DOWN, 3L, 3S | 高波动风险 |
| **法币配对** | USD, EUR, GBP, JPY | 不适合加密策略 |
| **稳定币** | USDT, USDC, BUSD, DAI | 波动性不足 |
| **粉丝代币** | ACM, PSG, JUV, CHZ | 高操纵风险 |
| **已知问题代币** | LUNA, FTT, SRM | 历史暴雷 |
| **隐私币** | XMR, ZEC | 监管风险 |

### 黑名单正则表达式

```json
{
  "exchange": {
    "pair_blacklist": [
      "(BNB)/.*",
      ".*(_PREMIUM|BEAR|BULL|HALF|HEDGE|UP|DOWN|[1235][SL])/.*",
      "(USD|EUR|GBP|JPY|AUD|CAD)/.*",
      "(USDT|USDC|BUSD|DAI|TUSD|USDN|USDP)/.*",
      "(LUNA|FTT|SRM|UST|USTC)/.*",
      "(ACM|PSG|JUV|CHZ|CITY|BAR|ATM)/.*",
      "(XMR|ZEC)/.*"
    ]
  }
}
```

### 正则表达式说明

| 模式 | 匹配示例 | 说明 |
|------|----------|------|
| `(BNB)/.*` | BNB/USDT | 任何BNB配对 |
| `.*BULL/.*` | ETHBULL/USDT | 杠杆多头代币 |
| `.*3L/.*` | BTC3L/USDT | 3倍杠杆代币 |
| `(USD)/.*` | USD/USDT | 法币配对 |

### 交易所特定黑名单文件

- `blacklist-binance.json` - Binance 专用
- `blacklist-kucoin.json` - Kucoin 专用
- `blacklist-okx.json` - OKX 专用

## 2.6 策略参数覆盖

### 通过 nfi_parameters 覆盖

```json
{
  "nfi_parameters": {
    "num_cores_indicators_calc": 4,
    "futures_mode_leverage": 3.0,
    "stop_threshold_spot": 0.10,
    "grinding_v1_max_stake": 1.0,
    "rebuy_mode_stake_multiplier": 0.35,
    "derisk_enable": true,
    "custom_fee_open_rate": 0.0,
    "custom_fee_close_rate": 0.0
  }
}
```

### 可覆盖的关键参数

| 参数 | 类型 | 说明 |
|------|------|------|
| `num_cores_indicators_calc` | int | 多核指标计算 |
| `futures_mode_leverage` | float | 期货杠杆 |
| `stop_threshold_spot` | float | 现货止损 |
| `stop_threshold_futures` | float | 期货止损 |
| `grinding_v1_max_stake` | float | 磨削最大头寸 |
| `rebuy_mode_stake_multiplier` | float | 回购仓位乘数 |
| `derisk_enable` | bool | 风险撤离开关 |

## 2.7 API 配置

### 交易所连接配置

```json
{
  "exchange": {
    "name": "binance",
    "key": "",
    "secret": "",
    "ccxt_config": {
      "enableRateLimit": true,
      "rateLimit": 200
    },
    "ccxt_async_config": {
      "enableRateLimit": true,
      "rateLimit": 200
    }
  }
}
```

### 代理配置（可选）

```json
{
  "exchange": {
    "ccxt_async_config": {
      "aiohttp_trust_env": true
    }
  }
}
```

环境变量设置：
```bash
export HTTP_PROXY="http://proxy:port"
export ALL_PROXY="socks5://user:pass@host:port"
```

### 订单配置

```json
{
  "unfilledtimeout": {
    "entry": 3,
    "exit": 2,
    "unit": "minutes"
  },
  "order_types": {
    "entry": "limit",
    "exit": "limit"
  }
}
```

## 2.8 持有交易支持

### 配置文件

创建 `user_data/nfi-hold-trades.json`：

```json
{
  "trade_ids": [1, 3, 7],
  "profit_ratio": 0.005
}
```

### 按交易对指定

```json
{
  "trade_pairs": {
    "BTC/USDT": 0.001,
    "ETH/USDT": -0.005
  }
}
```

### 参数说明

| 参数 | 说明 |
|------|------|
| `trade_ids` | 保持的交易ID列表 |
| `profit_ratio` | 全局利润比率阈值 |
| `trade_pairs` | 按交易对指定利润阈值 |
