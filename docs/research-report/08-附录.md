## 附录B：交易模式快速参考卡

### B.1 多头模式

| 模式 | 标签 | 适用场景 | 风险级别 | 利润目标 |
|------|------|----------|----------|----------|
| **Normal** | 1-13 | 温和趋势 | 低 | 1-5% |
| **Pump** | 21-26 | 快速突破 | 高 | 同标准 |
| **Quick** | 41-53 | 短期动量 | 中 | 1-3% |
| **Rebuy** | 61-62 | 区间盘整 | 中 | 分阶段 |
| **Rapid** | 101-110 | 高频交易 | 高 | 5% |
| **Grind** | 120 | 缓慢趋势 | 低 | 1.8-4.8% |
| **Scalp** | 161-163 | 超短期 | 高 | 0.5-1.5% |

### B.2 空头模式

| 模式 | 标签 | 适用场景 | 风险级别 | 利润目标 |
|------|------|----------|----------|----------|
| **Normal** | 501-502 | 温和下跌 | 低 | 1-5% |
| **Pump** | 521-526 | 高位回调 | 高 | 同标准 |
| **Quick** | 541-550 | 短期空头 | 中 | 1-3% |
| **Rebuy** | 561 | 区间重入 | 中 | 分阶段 |
| **Rapid** | 601-610 | 快速空头 | 高 | 5% |
| **Grind** | 620 | 分层空头 | 低 | 1.8-4.8% |
| **Scalp** | 661 | 超短空头 | 高 | 0.5-1.5% |

### B.3 模式选择决策树

```
市场分析
    │
    ├─ 强势趋势？
    │   ├─ 是 → Normal + Grind
    │   └─ 否 → 继续
    │
    ├─ 高波动突破？
    │   ├─ 是 → Pump
    │   └─ 否 → 继续
    │
    ├─ 区间盘整？
    │   ├─ 是 → Rebuy
    │   └─ 否 → 继续
    │
    ├─ 高频需求？
    │   ├─ 是 → Rapid / Scalp
    │   └─ 否 → Normal
    │
    └─ 默认 → Normal
```

## 附录D：常用命令速查

### D.1 配置相关

```bash
# 验证配置
freqtrade validate-config -c config.json

# 列出配置
freqtrade list-config -c config.json

# 列出所有配置
freqtrade list-config --all
```

### D.2 交易相关

```bash
# 启动干运行
freqtrade trade --strategy NostalgiaForInfinityX6 --dry-run -c config.json

# 启动实盘
freqtrade trade --strategy NostalgiaForInfinityX6 -c config.json

# 强制平仓
freqtrade forceexit --all -c config.json
```

### D.3 回测相关

```bash
# 基本回测
freqtrade backtesting \
  --strategy NostalgiaForInfinityX6 \
  --timerange 20240101-20250101 \
  -c config.json

# 多配置回测
freqtrade backtesting \
  --strategy NostalgiaForInfinityX6 \
  -c configs/trading_mode-futures.json \
  -c configs/pairlist-volume-binance-usdt.json \
  -c configs/exampleconfig.json \
  --timerange 20240101-20250101
```

### D.4 数据相关

```bash
# 下载数据
freqtrade download-data \
  --exchange binance \
  --pairs BTC/USDT ETH/USDT \
  --timeframes 5m 15m 1h 4h 1d \
  --timerange 20230101-

# 列出交易对
freqtrade list-pairs -c config.json --exchange binance
```

### D.5 Docker 相关

```bash

docker compose up -d

docker compose down

docker compose logs -f
```
